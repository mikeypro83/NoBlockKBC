    GNU C++ Class for non-blocking keyboard input in Linux Terminal (as well as Windows Console for compatibility)

    Project maintained by mikeypro83 Hosted on GitHub Pages — Theme by mattgraham

    General Info

    A simple c++ header file that contains a class with functions to perform simple keyboard input checking in a Linux Terminal as well as a Windows Console, to simplify prototype development on the two platforms.
